# Outreach Templates

> Pre-built message templates for common networking scenarios.
> Customize these with your own voice and proof points.

## Cold LinkedIn Connection Request (300 char max)

```
Hi [Name] -- I noticed [specific thing about their work/company].
I [your relevant proof point in 1 sentence].
Would love to chat about [specific topic] for 15 min. Open to connecting?
```

## Follow-up After Application

```
Hi [Name] -- I recently applied for the [Role] position at [Company].
I'm particularly excited about [specific thing from JD].
My background in [relevant experience] maps directly to what you're building.
Happy to share more context if helpful.
```

## Post-Interview Thank You

```
[Name] -- Thanks for the conversation today.
I especially enjoyed discussing [specific topic from interview].
It reinforced my interest in [what excited you about the role].
Looking forward to next steps.
```

## Warm Introduction Request

```
Hi [Mutual Contact] -- I'm exploring [Role type] roles and noticed
[Target Person] at [Company]. Given your connection, would you be
comfortable making an intro? I can send you a blurb to forward.
```
